
## Description

API for [React + TypeScript + Webpack - Stripe integration](https://github.com/let-none-survive/react-typescript-stripe)

## Installation

```bash
$ yarn install
$ cp .env.sample .env
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
